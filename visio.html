Methodology

My approach to process mapping follows these key steps:

  1. Discovery: Interviews with stakeholders and process owners to understand current workflows
  2. Documentation: Creation of "as-is" process maps to visualize current state
  3. Analysis: Identification of inefficiencies, bottlenecks, and opportunities
  4. Redesign: Development of "to-be" process maps that incorporate improvements
  5. Implementation: Support for rolling out new processes with clear visual guidance
  6. Measurement: Integration with performance data to track improvements

Featured Process Maps

Claims Processing Workflow

A comprehensive process map documenting the end-to-end claims journey from first notification of loss through to settlement. This map helped identify several redundant steps and approval bottlenecks.

Key Improvements Identified:

  • Reduced approval stages from 5 to 3, cutting processing time by 22%
  • Identified and eliminated duplicate data entry points
  • Created clearer handoff processes between departments
  • Implemented automated notifications at key process milestones

Finance Approval System

A cross-functional swim lane diagram showing the finance approval process across multiple departments. This visualization enabled stakeholders to see how their part of the process affected overall efficiency.

Excel Integration:

This diagram was linked to Excel data to show:

  • Average processing time at each stage (color-coded for quick identification of bottlenecks)
  • Volume of requests flowing through each path
  • Approval rates by department and manager level
  • Cost implications of delays at each stage

Excel-Visio Integration Innovation

One of my key innovations was developing a method to link Excel data directly to Visio diagrams, allowing for:

  • Dynamic process maps that update automatically when data changes
  • Color-coded process steps based on performance metrics
  • Quantitative analysis of process flow volumes
  • Time and cost tracking for each process step
  • Easier identification of improvement opportunities through data visualization

This integration has been adopted as a best practice within the organization and has led to significant improvements in process efficiency and data-driven decision making.
